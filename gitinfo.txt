1. git init -> создание нового файла настройки .git

2. git add <filename> -> добавление файла на загрузку
2.1 git add . -> добавить все файлы в папке на загрузку

3. git commit -m <message> -> подтверждает изменения в файлах и окончательно готовит их к загрузке

4. git remote add <link names> <sshlink> -> пишется один раз после git init, добавляет ссылку на репотизорий в конфигурацию

5. git push origin <branch_name> -> загрузить изменения, совершенные в git commit, на сайт

Для загрузки новых файлов повторить шаги (2, 3, 5)
